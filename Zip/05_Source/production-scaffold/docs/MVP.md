# MVP Scope

## Milestone 1

Persist the core operations loop:

1. Create booking.
2. List/search/filter bookings.
3. Assign driver by date and time slot.
4. Update pickup status.
5. Move No Show customer to later round.
6. Print driver job sheet.

## Deferred

- Full KPI dashboard.
- Advanced pivot analysis.
- Realtime updates.
- Excel export pipeline.
- Role-specific mobile views.
